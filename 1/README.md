# Задание 1
## Описание
Программа вычисляет сумму значений синуса для массива из 10⁷ элементов. Поддерживаются два типа данных: float  и double.

# Как выбрать тип массива

## Через опцию CMake
```bash
# Сборка с FLOAT (по умолчанию)
cmake .. && make && ./task1

# Сборка с DOUBLE
cmake .. -DUSE_DOUBLE=ON && make && ./task1
```

# Результаты тестирования

## Версия с float
```text
Sum: -0.213894307613373
```

## Версия с double
```text
Sum: 2.68623371349347e-07
```
